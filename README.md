Java Basics Programs

This repository contains simple Java basic programs written to understand core Java concepts. These programs are suitable for beginners and interview preparation.

Topics Covered

Addition Program – Demonstrates arithmetic operations.

Method / Function Calling – Shows how methods are defined and called from the main method.

Comments – Usage of single-line and multi-line comments in Java.

Data Types – Examples of primitive data types such as int, double, char, and boolean.

Local Variables – Variables declared inside methods and their scope.

Global Variables – Instance and static variables and how they are accessed.

Print Name Program – Simple program to print a name using System.out.println().

Purpose

To practice Java fundamentals

To understand syntax, scope, and structure

To build a strong base for Java interviews and advanced concepts

Technologies Used

Java

Eclipse / IntelliJ IDEA

How to Run

Clone the repository

Open in any Java IDE

Run the required .java file
